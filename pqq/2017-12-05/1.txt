1. WEB URL(html页面)----GET
- 首页：   
  url:/  
  说明：有注册登录的链接，可进行跳转
- 用户登录：
  url:/user/login
  说明：显示登录页面
- 用户注册：
  url:/user/register
  说明：显示注册页面
- 个人邮箱页面：
  url:/user/mailmain  
  说明：有三个tab页面 分别为发送邮件页面；收件箱；发件箱
  

2. WEB API(json数据交互)----POST
action = req.body.action

user
- 用户登录： /api/user
action=login
Method = post

- 用户注册:   /api/user
action=register
Method = post

-用户注销：  /api/user
action=logout
Method = post

mail :    /api/mail
发送邮件：action=create
未读邮件变已读：action=update
Method = post

- 读取=GET  
  /api/mail？receiver=当前用户  ----  这个为自己的收件箱  
  /api/mail? sender=当前用户    ----  这个为自己的发件箱  
  /api/mail? mailid=xxx          ---- 读取单个邮件
