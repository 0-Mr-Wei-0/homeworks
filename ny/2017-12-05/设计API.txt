Web URL
1.首页		 /
作用：显示网站首页

2.用户登录	/users/login
作用：用户登录

3.用户注册	/users/register
作用：用户注册

4.邮箱页面	/mails
作用：显示邮箱主页

5.写邮件		/mails/write
作用：写邮件
6.邮件列表	/mails/list
作用：显示用户邮箱的所有接收邮件

7.读邮件		/mails/list/id
作用：读具体的一封邮件


API
1.用户		users
读取用户		GET /users
读取具体用户	GET /users/id
读取用户第一页前50条	GET /users?page=1&limit=50
用户登录		
	url:/users
	method:POST
	data:action=login
	作用：用户登录
用户注册		
	url:/users
	method:POST
	data:action=register
	作用：用户注册
用户退出	
	url:/users
	method:POST
	data:action=loginout
	作用：用户退出

2.邮件		mails
写邮件	
	url:/mails
	method:POST
	data:action=write
	作用：写邮件
邮件列表		GET /mails/list
读具体邮件	GET /mails/list/id
