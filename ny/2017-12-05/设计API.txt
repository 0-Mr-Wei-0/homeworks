Web URL
1.首页		 /
作用：显示网站首页

2.用户登录	/user/login
作用：用户登录

3.用户注册	/user/register
作用：用户注册

4.邮箱页面	/mail
作用：显示邮箱主页

5.写邮件		/mail/write
作用：写邮件
6.邮件列表	/mail/list
作用：显示用户邮箱的所有接收邮件

7.读邮件		/mail/list/id
作用：读具体的一封邮件


API
1.用户		user
读取用户		GET /user
读取具体用户	GET /user/id
读取用户第一页前50条	GET /user?page=1&limit=50
用户登录		
	url:/user
	method:POST
	data:action=login
	作用：用户登录
用户注册		
	url:/user
	method:POST
	data:action=register
	作用：用户注册
用户退出	
	url:/user
	method:POST
	data:action=loginout
	作用：用户退出

2.邮件		mail
写邮件	
	url:/mail
	method:POST
	data:action=write
	作用：写邮件
邮件列表		GET /mails/list
读具体邮件	GET /mails/list/id
