一．Web URL （返回HTML页面）
1. 用户登录      url:/user/login         显示登陆界面
2. 用户注册      url:/user/register     显示注册界面
3. 邮件主界面   url:/mail          显示邮件主界面
4. 写邮件         url:/mail/write         显示写邮件界面，以及发送
5. 读邮件         url:/mail/read         显示读邮件界面

二．Web API （返回JSON数据）
1. / users 用户
       GET  /users          读取用户列表
              /users/:id     读取单个用户信息
       POST /users/action = login       登录
                /users/action = register    注册
2. /mail 邮件
      GET    /mails    读取邮件列表
      POST /mails/action = send      发送邮件
