一．Web URL

1.系统主页面
GET  /	

2.用户登录界面
GET  /users/login

3.用户注册界面
GET  /users/register

4.邮件列表界面
GET  /mails/list

5.发送邮件界面
GET  /mails/send


二．Web API

1.用户

POST  /api/users/
Data: action=login
用户登录验证

POST  /api/users/
Data:action=register
用户注册验证


GET  /api/users/
获取用户列表

GET  /api/users/:id
获取单个用户信息

2.邮件

GET  /api/mails/
获取邮件列表

GET  /api/mails/:id
获取单个邮件


POST  /api/mails/
data:action=send
发送邮件
