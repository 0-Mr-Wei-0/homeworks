Web URL：
1.用户登录页面：/users/login
2.用户注册页面：/users/register
3.首页：/
4.用户查看信息页面：/users/info
5.邮件编写页面：/mails/write
6.收件箱页面：/mails/read


Web API:
1.用户注册：
url：/users
method：POST
data：action=register
作用：接受用户输入的数据，提示注册是否成功。

2.用户登录
url:/users
method:POST
data:action=login
作用：接受用户输入数据，判断用户名是否存在，提示登录是否成功。

3.编写邮件
url:/mails
method:POST
data:action=write
作用：接受用户提交的邮件内容，判断目标邮箱是否存在，写入数据库。

4.查看邮件列表：
url:/mails
method:GET
作用：向用户返回邮件的列表信息。

5.退出登录
url:/users
method:POST
data:action=logout
作用：用户退出登录。

6.查看用户列表：
url:/users
method:GET
作用：向用户返回用户列表信息。

7.查看单个用户信息：
url:/users/:id
method:GET
作用：返回某个用户id的具体用户信息
