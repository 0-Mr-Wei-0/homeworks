1.首页
	url:/
	作用：首页显示跳转登陆和注册链接
2.用户登陆
	url：/user/login
	作用：实现用户登陆
3.用户注册
	url:/user/register
	作用：实现用户注册
4.邮箱主页
	url:/mail
	作用：显示跳转邮件各个功能的链接	
5.发送邮件
	url:/mail/send
	作用：实现发送邮件功能
6.读取邮件列表
	url:/mail/list
	作用：显示用户接受的邮件列表
7.读取邮件
	url:/mail/list/read
	作用：读取邮件的详细信息

API
１.用户 User
	用户登陆 action = login
	用户注册 action = register
2.邮件 Mail
	发送邮件 action = send
	读取邮件列表 action = list
	读取邮件 action = read
