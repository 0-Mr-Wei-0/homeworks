1.Web URL
  (1)用户登录
     url:/user/login
     作用：用户登录页面
   (2)用户注册
     url:/user/regist
     作用：用户注册页面
   (3)主页面
     url:/user/home
     作用：主页面
   (4)邮件页面
     url:/user/mail
     作用：邮件页面

2.Web API
   (1)用户
      url: /users
      action = [login,regist,update]
   (2)邮件
      url:/mails
      action = [receive,write,send]
      //write ---写邮件 
      //send ----发送邮件
      //receive ---接收邮件
